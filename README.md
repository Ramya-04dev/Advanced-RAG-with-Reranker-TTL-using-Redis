📚 Advanced RAG with Reranker (TTL using Redis)
🔍 Project Overview
This project implements an Advanced Retrieval-Augmented Generation (RAG) system using:
FAISS for semantic search
CrossEncoder for reranking
Redis for caching with TTL
TinyLlama for local summarization
⚙️ Features
🔎 Semantic search using embeddings
🎯 Improved relevance using reranker
⚡ Faster responses with Redis caching (TTL)
🤖 Local LLM-based summarization
🛠️ Technologies Used
Python
LangChain
FAISS
Redis
Transformers (CrossEncoder)
TinyLlama
📂 Files
reranker.py → Main application code
requirements.txt → Dependencies
🚀 How to Run
Clone the repository:
git clone https://github.com/Ramya-04dev/Advanced-RAG-with-Reranker-TTL-using-Redis.git
Install dependencies:
pip install -r requirements.txt
Run the project:
python reranker.py
